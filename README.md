# Noxer-app - Приложение для онлайн-магазина(тестовое задание)

В рамках тестового задания надо было сверстать главную страницу SPA-приложения по макету, реализовать подгрузку и динамический поиск и автоматический поиск товаров по фразе "Часто ищут".
